🍎 Low Hanging Fruit: Quick TPL Alignment Wins

✅ Easy Fix #1: Root Font Family

CURRENT
:root {
  font-family: 'Clear Sans', 'Helvetica Neue', Arial, sans-serif;
}
TPL ALIGNED
:root {
  font-family: 'nyt-franklin', helvetica, arial, sans-serif;
}
Impact: Instantly aligns all interface text with TPL standards

✅ Easy Fix #2: Title Token Alignment

CURRENT
.title {
  font-size: 36px; // mobile
  @media (min-width: 760px) { font-size: 44px; }
  @media (min-width: 992px) { font-size: 50px; }
}
TPL ALIGNED
.title {
  font: 600 2rem/1.2 nyt-franklin; // Title/32
  @media (min-width: 481px) { font-size: 3rem; } // Title/48
}
Impact: Uses TPL tokens and breakpoints

✅ Easy Fix #3: Stats Label Styling

CURRENT
.stat-label {
  font-size: 12px;
  color: #787c7e;
  text-transform: uppercase;
}
TPL ALIGNED
.stat-label {
  font: 500 0.6875rem/1.25 nyt-franklin; // Label/Regular
  letter-spacing: 0.1em;
  text-transform: uppercase;
}
Impact: Matches TPL label styling exactly

✅ Easy Fix #4: Font Weight Standardization

CURRENT
font-weight: bold; // Inconsistent
font-weight: 700; // Mixed usage
font-weight: 600; // Various patterns
TPL ALIGNED
font-weight: 600; // semiBold (Title)
font-weight: 500; // medium (Text)
font-weight: 800; // extraBold (Label/Emphasis)
Impact: Consistent weight usage across all text

✅ Easy Fix #5: Line Height Consistency

CURRENT
line-height: 38px; // Fixed pixel values
line-height: 1.045em; // Mixed units
line-height: 52px; // Inconsistent approach
TPL ALIGNED
line-height: 1.2; // Titles
line-height: 1.3; // Text
line-height: 1.25; // Labels
Impact: Systematic line-height ratios

✅ Easy Fix #6: Breakpoint Alignment

CURRENT
@media (min-width: 760px) { ... }
@media (min-width: 992px) { ... }
TPL ALIGNED
@media (min-width: 481px) { ... } // TPL medium
@media (min-width: 1025px) { ... } // TPL large
Impact: Consistent breakpoints across NYT products

🎯 Implementation Priority

These changes can be implemented incrementally without affecting game functionality. Start with the root font family change for immediate visual alignment with TPL standards.

Estimated effort: 2-4 hours • Risk: Low • Impact: High visual consistency
