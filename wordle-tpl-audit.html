Typography System Replacement

Design Tokens High Priority

Current Implementation

Source: index.scss (lines 12-75)

.tile { font-family: 'nyt-franklin'; font-size: 2rem; font-weight: bold; } .heading { font-family: 'nyt-franklin'; font-weight: 700; font-size: 16px; letter-spacing: 0.5px; }

TPL Alternative

<Typography variant="label/1" as="div"> Game Content </Typography> <Typography variant="headline/4" as="h2"> Modal Heading </Typography>

Impact: Eliminate 76 lines of custom font declarations and ensure consistent typography across all NYT products.

  • Theme Support: Dark mode and colorblind mode need TPL compatibility

Migration Complexity

  • Performance Monitoring: Monitor for any performance regressions during migration
  • Fallback Strategy: Plan for quick rollback if issues arise
